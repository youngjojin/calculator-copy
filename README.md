# Lecture_test1-open-source-practice
Example for lecture of JUN OS
This repository is used for learning open source contribution workflow.

Students will practice:

- Git
- Issue
- Branch
- Pull Request
- Code Review

** 학생 전용: Pull Request 제출 가이드 **
과제를 마친 후 GitHub에서 [Create pull request] 버튼을 누를 때, 아래 형식을 꼭 지켜주세요.

1. PR 제목 형식
[과제1] 학번 / 이름 / Add division function
(예: [과제1] 202401234 / 홍길동 / Add division function)

2. PR 본문 내용 (Template)
아래 내용을 복사해서 본문에 작성하세요.

Markdown
## 과제 제출 정보
- **이름:** 홍길동
- **학번:** 202401234
- **Fork 저장소 주소:** (여기에 본인 저장소 링크 붙여넣기)

## 수행 내용 확인
- [x] `feature-division` 브랜치 생성 완료
- [x] `divide` 함수 구현 완료
- [x] `test_divide` 테스트 코드 작성 및 통과 확인
